## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Prisma

This project uses [Prisma](https://www.prisma.io/) to connect to a MongoDB database. You can find the Prisma schema in the `prisma/schema.prisma` file.

To apply the schema to your database, run the following command:

```bash
npx prisma db push
```
